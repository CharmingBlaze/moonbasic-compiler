# Axis Commands

D-pad / axis helpers for gamepad input.

## Commands

### `AXIS.DPADY(gamepadId)` 

Returns the D-pad vertical axis value for gamepad `gamepadId`: `-1.0` = up, `1.0` = down, `0.0` = centred.
