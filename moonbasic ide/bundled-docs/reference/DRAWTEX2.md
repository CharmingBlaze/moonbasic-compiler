# DrawTex2 Commands

`DRAWTEX2.*` commands are documented in [DRAWTEX.md](DRAWTEX.md) under the **DRAWTEX2 — Simple Texture Draw** section.

## Quick Reference

| Command | Description |
|---|---|
| `DRAWTEX2.SETTEXTURE(handle, tex)` | Bind a texture |
| `DRAWTEX2.POS(handle, x, y)` | Set draw position |
| `DRAWTEX2.COLOR(handle, r, g, b, a)` | Set color tint |
| `DRAWTEX2.DRAW(handle)` | Draw this frame |
| `DRAWTEX2.FREE(handle)` | Free handle |
