# DrawTexRec Commands

`DRAWTEXREC.*` commands are documented in [DRAWTEX.md](DRAWTEX.md) under the **DRAWTEXREC — Source Rect Crop** section.

## Quick Reference

| Command | Description |
|---|---|
| `DRAWTEXREC.SETTEXTURE(handle, tex)` | Bind texture atlas |
| `DRAWTEXREC.SRC(handle, x, y, w, h)` | Source crop rect |
| `DRAWTEXREC.POS(handle, x, y)` | Screen position |
| `DRAWTEXREC.COLOR(handle, r, g, b, a)` | Color tint |
| `DRAWTEXREC.DRAW(handle)` | Draw this frame |
| `DRAWTEXREC.FREE(handle)` | Free handle |
