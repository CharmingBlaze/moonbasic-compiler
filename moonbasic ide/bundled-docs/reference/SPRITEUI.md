# SpriteUI Commands

`SPRITEUI.*` commands are documented in [SPRITEBATCH.md](SPRITEBATCH.md) under the **SpriteUI** section.

## Quick Reference

| Command | Description |
|---|---|
| `SPRITEUI.CREATE(sprite, x, y)` | Create a screen-space UI sprite element |
| `SPRITEUI.DRAW(handle, x, y)` | Draw at position |
| `SPRITEUI.FREE(handle)` | Free handle |

## Extended Command Reference

| Command | Description |
|--------|-------------|
| `SPRITEUI.MAKE(...)` | Deprecated alias of `SPRITEUI.CREATE`. |
