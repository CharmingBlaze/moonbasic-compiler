# Music Commands

Music stream handle management. For loading and playback see [AUDIO.md](AUDIO.md).

## Commands

### `MUSIC.FREE(musicHandle)` 

Frees a music stream handle previously loaded with `AUDIO.LOADMUSIC` or `MUSIC.LOAD`.
