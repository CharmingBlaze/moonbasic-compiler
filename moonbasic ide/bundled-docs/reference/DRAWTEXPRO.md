# DrawTexPro Commands

`DRAWTEXPRO.*` commands are documented in [DRAWTEX.md](DRAWTEX.md) under the **DRAWTEXPRO — Full Pro Control** section.

## Quick Reference

| Command | Description |
|---|---|
| `DRAWTEXPRO.SETTEXTURE(handle, tex)` | Bind texture |
| `DRAWTEXPRO.SRC(handle, x, y, w, h)` | Source rect |
| `DRAWTEXPRO.DST(handle, x, y, w, h)` | Destination rect |
| `DRAWTEXPRO.ORIGIN(handle, ox, oy)` | Rotation origin |
| `DRAWTEXPRO.ROT(handle, degrees)` | Rotation angle |
| `DRAWTEXPRO.COLOR(handle, r, g, b, a)` | Color tint |
| `DRAWTEXPRO.DRAW(handle)` | Draw this frame |
| `DRAWTEXPRO.FREE(handle)` | Free handle |
