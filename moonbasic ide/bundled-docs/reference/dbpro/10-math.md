# DBPro — Math

moonBASIC uses **`SIN`**, **`COS`**, … and **`MATH.*`** overloads — [MATH.md](../MATH.md).

| DBPro | moonBASIC | Notes |
|-------|-----------|--------|
| **ABS(x)** | ✓ **`Abs()`** | |
| **SQRT(x)** | ✓ **`Sqrt()`** / `Sqr()` | |
| **SIN(x)** / **COS(x)** / **TAN(x)** | ✓ **`Sin()`** / `Cos()` / `Tan()` | Radians. |
| **ASIN(x)** / **ACOS(x)** / **ATAN(x)** | ✓ **`Asin()`** / `Acos()` / `Atan()` | Radians. |
| **INT(x)** | ✓ **`Int()`** / `Fix()` | |
| **EXP(x)** | ✓ **`Exp()`** | |
| **LOG(x)** | ✓ **`Log()`** | |
| **RND(limit)** | ✓ **`Rnd()`** | |
| **RANDOMIZE(seed)** | ✓ **`Randomize()`** | |
| **RNDSEED(seed)** | ✓ **`RndSeed()`** | |

**Note:** DBPro **SQR** sometimes meant “square” in other BASICs; moon **`SQR`** is documented as **square** in some manifests — use **`SQRT`** for √x.
