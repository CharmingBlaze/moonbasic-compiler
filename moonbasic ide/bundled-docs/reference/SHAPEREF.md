# ShapeRef Commands

Handle-method free for Jolt collision shape handles created via `SHAPE.*`.

## Commands

### `SHAPEREF.FREE(shapeHandle)` 

Destroys a Jolt collision shape handle. Alias of `SHAPE.FREE`.
